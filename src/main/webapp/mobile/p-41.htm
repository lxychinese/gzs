Restlet 输出日志说明

日期：2015-07-25   来源：垄上工作室

Restlet在接受请求并处理后，会在控制终端输入的信息。

 

下面是输出的内容举例：

2008-01-21 15:33:09 172.20.82.225 - 127.0.0.1 80 POST /WebRoot/rest/controller - 200 6 65 2953 http://172.20.82.225  Noelios-Restlet-Engine/1.1.m3 -

 

 

说明

含义

参考值

Date (YYYY-MM-DD)

请求日期

2008-01-21

Time (HH:MM:SS)

请求时间

15:33:09

Client address (IP)

客户端IP地址

172.20.82.225

Remote user identifier (see RFC 1413)

客户端身份

-

Server address (IP)

服务端IP地址

127.0.0.1

Server port

服务端口

80

Method (GET|POST|...)

请求方法

POST

Resource reference path (including the leading slash)

请求路径 (Query)

/myapp/rest/myresource

Resource reference query (excluding the leading question mark)

请求地址携带的内容

-

Response status code

响应状态代码

200

Number of bytes sent

(Returned Size)响应内容长度

6

Number of bytes received

(Received Size) 收到内容长度

65

Time to serve the request (in milliseconds)

(Duration)处理调用时长

2953

Host reference

(Host Reference)请求主机地址

http://172.20.82.225

Client agent name

(Agent)客户端代理名称

Noelios-Restlet-Engine/1.1.m3

Referrer reference

(Referrer) 来源页面地址

-

 

参考网址：

 

http://www.w3.org/TR/WD-logfile.html
http://java.sun.com/j2se/1.5.0/docs/api/java/util/logging/package-summary.html
